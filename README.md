# To-Do List Application

## Overview
This To-Do List Application allows users to manage their tasks efficiently. Users can add, edit, delete, and mark their tasks as completed.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as complete/incomplete
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation
1. Clone the repository using `git clone`.
2. Open `index.html` in your web browser to start using the app.

## Usage
- To add a new task, enter the task details and click 'Add'.
- To edit a task, click on the task and modify the details.
- To delete a task, click the 'Delete' button next to the task.
